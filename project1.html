Description

For my CNT 4714 course, I developed a Java Swing application simulating an online store. The program allows users to search inventory, add items to a shopping cart, apply quantity-based discounts, and complete checkout. Each transaction is logged into a file for record-keeping. This project emphasized event-driven programming and object-oriented design.

Skills Learned

  • Java GUI development with Swing
  • Object-Oriented Programming (OOP)
  • Event-driven design
  • File I/O for inventory and transaction management
  • Problem-solving and debugging
